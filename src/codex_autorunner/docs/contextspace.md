# Contextspace

Contextspace is repo-local durable context for agent work.

Common docs:

- `active_context.md`: current working context and links
- `spec.md`: requirements, target behavior, and acceptance criteria
- `decisions.md`: durable decisions and rationale

Keep transient details in active context. Move durable guidance to the closest applicable `AGENTS.md`.

PMA has separate hub-level memory under `.codex-autorunner/pma/docs/`. Do not copy PMA docs into repo contextspace to make relative paths work.
