# Glossary

Hub: the CAR control-plane root containing `.codex-autorunner/config.yml`.

Repo: a registered git checkout managed by the hub.

Worktree: a branch checkout associated with a repo.

PMA: Project Management Agent, the hub-level coordinator.

Managed thread: a CAR-tracked agent conversation tied to a resource.

Ticket flow: ordered execution over repo-local ticket files.

Dispatch: a paused run message or user-attention item.

FileBox: inbox/outbox folders for file handoff.

Contextspace: repo-local durable context docs.
